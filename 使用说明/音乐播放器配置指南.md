# 博客音乐播放器配置指南

## 概述

博客使用 APlayer 播放器 + PJAX 无刷新切换，直接播放存放在仓库中的本地音频文件，不依赖任何外部 API。
切换页面时音乐不会中断，持续播放。

## 文件结构

```
js/music-pjax.js        # 播放器 + PJAX 核心脚本（歌曲列表在此文件中配置）
medias/music/            # 音频文件目录
├── yongchun_dj.m4a      # 音频文件（支持 mp3、m4a、flac、wav 等格式）
├── yongchun_dj.jpg      # 封面图片（可选）
└── ...
.nojekyll                # 禁用 Jekyll 构建（确保 js 文件正常加载）
```

## 添加新歌曲

### 第一步：准备音频文件

将音频文件（mp3/m4a/flac 等）放入 `medias/music/` 目录。

### 第二步：编辑 js/music-pjax.js

找到文件顶部的 `musicList` 数组，添加一条记录：

```javascript
var musicList = [
    {
        name: '别辜负眼前季节',       // 歌曲名
        artist: '七朵组合',           // 歌手
        url: '/medias/music/yongchun_dj.m4a',   // 音频文件路径
        cover: '/medias/music/yongchun_dj.jpg'   // 封面图片路径（可选）
    },
    {
        name: '新歌名称',
        artist: '歌手名',
        url: '/medias/music/新文件.mp3',
        cover: '/medias/music/新封面.jpg'
    },
];
```

### 第三步：推送到 GitHub

```bash
git add medias/music/ js/music-pjax.js
git commit -m "feat: 添加新歌曲"
git push
```

等待 1-2 分钟 GitHub Pages 部署完成后刷新页面即可。

## 从B站视频提取音频

可以通过 Python 脚本从 B 站视频中提取音频，并自动同步视频标题到播放器配置。

### 使用方法

将以下脚本保存为 `download_bilibili_audio.py`，修改 `bvid` 和 `output_name` 后运行：

```python
import json
import urllib.request
import os
import re

# ============ 配置区 ============
bvid = "BV1veFMzwEqe"        # B站视频的BV号
output_name = "yongchun_dj"   # 输出文件名（不含扩展名）
output_dir = "medias/music"    # 输出目录
artist = ""                    # 歌手名（留空则使用UP主名称）
auto_update_js = True          # 自动将歌曲写入 js/music-pjax.js
# ================================

headers = {
    "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64)",
    "Referer": "https://www.bilibili.com"
}

# 1. 获取视频信息
print(f"正在获取视频信息: {bvid}")
req = urllib.request.Request(
    f"https://api.bilibili.com/x/web-interface/view?bvid={bvid}",
    headers=headers
)
with urllib.request.urlopen(req) as r:
    data = json.loads(r.read())

if data['code'] != 0:
    print(f"错误: {data.get('message', '未知错误')}")
    exit(1)

info = data['data']
aid = info['aid']
cid = info['cid']
title = info['title']
uploader = info.get('owner', {}).get('name', '未知')
duration = info['duration']
if not artist:
    artist = uploader
print(f"标题: {title}")
print(f"UP主: {uploader}")
print(f"时长: {duration // 60}分{duration % 60}秒")

# 2. 获取音频流地址
print("正在获取音频流...")
req2 = urllib.request.Request(
    f"https://api.bilibili.com/x/player/playurl?avid={aid}&cid={cid}&fnval=16&qn=30216",
    headers=headers
)
with urllib.request.urlopen(req2) as r:
    pdata = json.loads(r.read())

dash = pdata['data'].get('dash')
if not dash or not dash.get('audio'):
    print("错误: 无法获取音频流")
    exit(1)

audio = dash['audio'][0]
audio_url = audio['baseUrl']

# 3. 下载音频
os.makedirs(output_dir, exist_ok=True)
out_path = os.path.join(output_dir, f"{output_name}.m4a")

print(f"正在下载音频到: {out_path}")
req3 = urllib.request.Request(audio_url, headers=headers)
with urllib.request.urlopen(req3) as resp:
    with open(out_path, 'wb') as f:
        total = 0
        while True:
            chunk = resp.read(65536)
            if not chunk:
                break
            f.write(chunk)
            total += len(chunk)

size = os.path.getsize(out_path)
print(f"下载完成! 文件大小: {size / 1024 / 1024:.1f}MB")

# 4. 下载封面
pic_url = info.get('pic', '')
if pic_url:
    cover_path = os.path.join(output_dir, f"{output_name}.jpg")
    print(f"正在下载封面到: {cover_path}")
    if pic_url.startswith('//'):
        pic_url = 'https:' + pic_url
    req4 = urllib.request.Request(pic_url, headers=headers)
    with urllib.request.urlopen(req4) as resp:
        with open(cover_path, 'wb') as f:
            f.write(resp.read())
    print("封面下载完成!")

# 5. 自动写入 js/music-pjax.js（标题自动同步B站视频标题）
js_path = 'js/music-pjax.js'
if auto_update_js and os.path.exists(js_path):
    with open(js_path, 'r', encoding='utf-8') as f:
        js = f.read()

    # 转义标题中的单引号
    safe_title = title.replace("'", "\\'")
    safe_artist = artist.replace("'", "\\'")

    new_entry = (
        f"        {{\n"
        f"            name: '{safe_title}',\n"
        f"            artist: '{safe_artist}',\n"
        f"            url: '/{output_dir}/{output_name}.m4a',\n"
        f"            cover: '/{output_dir}/{output_name}.jpg'\n"
        f"        }},\n"
    )

    # 在 musicList 数组中插入新歌曲，同时避免重复
    if f"/{output_name}.m4a" in js:
        # 已存在，替换同名条目
        pattern = r"        \{[^}]*/" + re.escape(output_name) + r"\.m4a[^}]*\},?\n"
        js = re.sub(pattern, new_entry, js)
        print(f"已更新 js/music-pjax.js 中的 {output_name} 条目（标题同步为: {title}）")
    else:
        # 不存在，插入新条目
        js = js.replace(
            "var musicList = [\n",
            "var musicList = [\n" + new_entry
        )
        print(f"已添加新歌曲到 js/music-pjax.js（标题: {title}）")

    with open(js_path, 'w', encoding='utf-8') as f:
        f.write(js)

print(f"""
====================================
完成! 歌曲信息:
  标题: {title}（自动同步自B站视频标题）
  歌手: {artist}
  文件: /{output_dir}/{output_name}.m4a

下一步: git add . && git commit -m "feat: 添加 {title}" && git push
====================================
""")
```

### 运行脚本

```bash
python download_bilibili_audio.py
```

### 注意事项

- BV 号从 B 站视频链接中获取，例如 `bilibili.com/video/BV1veFMzwEqe/` 中的 `BV1veFMzwEqe`
- 建议选择时长较短的单曲视频，避免下载整段循环歌单（文件过大）
- GitHub 单文件大小限制为 100MB，一般单曲音频不会超过
- 下载的音频格式为 m4a（AAC 编码），主流浏览器均支持播放

## PJAX 无刷新切换

博客已启用 PJAX（PushState + AJAX），实现以下效果：

- **音乐不中断**：切换页面时音乐持续播放，不会暂停或重新开始
- **播放状态保持**：导航到新页面后自动恢复播放状态
- **页面组件自动重初始化**：切换页面后自动重新初始化代码高亮、图片灯箱、TOC 目录等

PJAX 相关代码位于 `js/music-pjax.js` 中，无需额外配置。

---

## 刷新页面恢复播放

播放器会自动保存播放状态到浏览器的 localStorage，刷新页面后自动恢复：

- **保存内容**：当前歌曲、播放进度、播放/暂停状态、音量
- **保存时机**：播放/暂停时实时保存，同时每 2 秒定期保存进度
- **恢复行为**：刷新后自动跳转到上次的歌曲和进度位置，尝试继续播放
- **浏览器限制**：部分浏览器可能阻止自动播放，此时在页面任意位置点击一下即可恢复

无需额外配置，`js/music-pjax.js` 已内置此功能。

## 播放器配置说明

播放器相关配置位于 `js/music-pjax.js` 中的 `new APlayer({...})` 部分：

| 参数 | 说明 | 当前值 |
|------|------|--------|
| `fixed` | 固定在页面底部 | `true` |
| `autoplay` | 自动播放 | `false` |
| `theme` | 主题颜色 | `#42b983` |
| `loop` | 循环模式：`all` 全部 / `one` 单曲 / `none` 不循环 | `all` |
| `order` | 播放顺序：`list` 顺序 / `random` 随机 | `list` |
| `volume` | 默认音量（0~1） | `0.7` |
| `listFolded` | 歌曲列表默认折叠 | `true` |

更多 APlayer 配置参考：https://aplayer.js.org/#/home
