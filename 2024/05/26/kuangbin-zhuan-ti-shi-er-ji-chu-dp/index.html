4546. 最大和加强加强版

题目描述

给定一个长度为 $n$ 的整数序列 $S_1,S_2,…,S_n$。

我们规定函数 $sum(i,j) = S_i+…+S_j$（$1 \le i \le j \le n$）。

现在，给定一个整数 $m$，你的任务是找到 $m$ 个数对 $(i,j)$，使得 $sum(i_1,j_1)+sum(i_2,j_2)+sum(i_3,j_3)+…+sum(i_m,j_m)$ 的和尽可能大（$i_x \le i_y \le j_x$ 或 $i_x \le j_y \le j_x$ 是不允许的）。

请你输出这个最大可能和。

输入格式

输入包含多组测试数据。

每组数据占一行，首先包含两个整数 $m,n$，随后包含 $n$ 个整数 $S_1,S_2,…,S_n$。

输出格式

每组数据输出一行，一个整数，表示最大可能和。

数据范围

每个输入最多包含 $10$ 组数据。
$1 \le n \le 10^6$,
$1 \le m \le 1000$,
每个输入的所有数据的 $n \times m$ 相加之和不超过 $5 \times 10^7$。
$-32768 \le S_x \le 32767$。

输入样例：

1 3 1 2 3
2 6 -1 4 -2 3 -2 3

输出样例：

6
8

题解

dp[i][j] : 前j个数字分成i组而且以j为结尾的最大和
dp[i][j] = max(dp[i-1][j-1],dp[i][j-1])+a[j]; // 分两种情况把a[j]成为新的一组 或 让a[j]放到原来的组中 而且这样不以j结尾的组就在 <j 的组里面被包含了，所以分类完全覆盖所有情况。
发现可以把j这一维优化掉，因为
直接优化成 dp[i] = max(dp[i-1],dp[i])+a[j];
但是再我们求i的时候dp[i]还是上一轮的，但是由于dp[i-1]已经被我们更新掉了，所以我们需要再使用一个数组把上一轮的dp[i-1]保存下来到这一轮使用。
所以再优化成dp[i] = max(dp2[i-1],dp[i])+a[j];
而dp2[i] = max(dp2[i],dp[i]);

for j : 1-m 
    dp[j]=dp2[j]=-inf

for j : 1 - n
    for i : 1 - m
        dp[i] = max(dp[i],dp2[i-1])+a[j];
    for i : 1 - m
        dp2[i] = max(dp2[i],dp[i]); // 为什么这个地方要取最大值呢 这就是因为我们要把所有的情况都包含如果不取最大值直接让dp2[i]=dp[i]的话我们只能包含以j结尾的，而漏掉了以j前面结尾的部分，而这部分正是dp2[i],它包含了小于i的其他结尾的最大值。
ans = max(dp[m],dp2[m]);

Code

#include<bits/stdc++.h>
using namespace std;
#define int long long
const int N = 1e6+2,M=1100;
int a[N],dp[M],dp2[M];
int n,m;
signed main()
{
    while(~scanf("%lld %lld",&m,&n))
    {
        for(int i=1;i<=n;i++)   scanf("%lld",&a[i]);
        for(int i=1;i<=m;i++)   dp[i]=dp2[i]=-1e10;

        for(int j=1;j<=n;j++)
        {
            for(int i=1;i<=m;i++)
                dp[i]=max(dp[i],dp2[i-1])+a[j];
            for(int i=1;i<=m;i++)
                dp2[i]=max(dp[i],dp2[i]);
        }
        printf("%lld\n",max(dp[m],dp2[m]));
    }
    return 0;
}
